Python Network Scanning Tool

A command-line tool I built to practice networking fundamentals by performing host discovery, port scanning, and basic service enumeration.

What It Does

  • Scans a target or subnet for live hosts
  • Checks common ports (or a user-defined range)
  • Outputs results cleanly for review
Tech Used
  • Python
  • Sockets
  • Optional: Nmap / Scapy integrations
What I Learned
  • TCP/IP behavior in real scans
  • Timeouts, retries, error handling
  • How to format output for usability
