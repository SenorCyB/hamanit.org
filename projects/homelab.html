Virtualized IT & Security Home Lab

A VMware-based lab environment I use to practice Windows Server, Active Directory, networking, and security fundamentals in a safe sandbox.

Goal

Build a realistic environment to strengthen my skills in system administration and security by configuring services, intentionally breaking things, and troubleshooting them end-to-end.

Environment
  • VMware virtual network + multiple VMs
  • Windows Server (AD DS, DNS)
  • Windows client(s)
  • Linux servers / Kali for tooling
What I Implemented
  • Users, groups, permissions, file access
  • DNS concepts + troubleshooting
  • Basic hardening + baseline configs
  • Controlled misconfigs + remediation
